 교수님 피드백 정리 리스트 (항목별 대응 포함)

1. 캐릭터 & 배경

[문제] 캐릭터와 배경이 따로 노는 느낌, 퀄리티 이질감
[개선] 배경 쉐이딩 정리, 캐릭터 폴리 수 낮춰 통일감 조정

[문제] 로우폴리 배경 각짐 문제
[개선] 폴리브릿지 스타일 참고하여 배경 폴리 수 조절

 2. 애니메이션 & 피드백

[문제] 로봇 애니메이션 자연스럽지 않음
[개선] 달리기 이펙트 2D → 3D 전환, 타이밍/속도 리밸런싱

[문제] 점프 애니메이션이 앞으로 밀리는 느낌
[개선] 풍선 유무에 따라 점프물리 분리 적용 (정점 도달 후 낙하)

[문제] 점프 모션 애니메이션 길고 타이밍 안 맞음
[개선] 애니메이션 분할 삽입 또는 타이밍 수치 수정 고려

 3. 기믹 

[문제] 기믹 연출 부족
[개선] 망치, 타일, 풍선 등에 이펙트 및 사운드 추가

[문제] 기믹이 단조롭고 다양성 부족
[개선] 바람 강화 / 움직이는 타일 / 무중력 / 레이저 등 추가

[문제] 위치 정보 없음 (유저가 어디 있는지 모름)
[개선] 고도 UI, 슬라이드 게이지 등 시각적 안내 강화

 4. UI/UX
[문제] UI 시인성 부족 (배경에 묻힘)
[개선] 반투명 배경 박스 적용, 위치 및 구성 통일성 유지

[문제] 인벤토리/슬롯 UI 비통일 (슬라이드/아이콘 다름)
[개선] 마리오카트 레퍼런스 참고해 UI 일관성 강화

[문제] 클릭 반응 없음
[개선] 버튼 클릭 시 스케일 애니메이션 / 사운드 피드백 적용

 5. 레벨 디자인
[문제] 레벨링이 어려움 / 흐름이 불분명
[개선] 스테이지별 튜토리얼 → 도전 → 종합 구조 설계 (총 4스테이지)

[문제] 맵에서 흐름이 끊김, 흐름 파악 어려움
[개선] 플로우 차트 설계, 각 구간의 목적 분명히 구분

 6. 보상/동기 부여
[문제] 보상 구조 불분명
[개선] 업적 → 칭호/스킨 지급 시스템 구축

[문제] 유저가 다시 시도할 이유 부족
[개선] 세이브존, 리스폰 연출, 업적 UI 추가로 반복 유도

 7. 기타
[문제] 스토리/연출 전달 약함
[개선] 클리어 시 연출 강화, 사운드 및 대사 포함 고려

[요청] 플로우 차트는 길면 여러 페이지로 나눠 구성
[조치] 전체 UI 플로우 차트는 슬라이드 구조로 분리 가능
